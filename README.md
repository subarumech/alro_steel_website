# Alro Steel Website

Modernized homepage for Alro Steel - America's Premier Metals and Plastics Service Center.

## Features

- Modern, clean design with minimalist aesthetic
- Responsive layout (mobile-first approach)
- Hero section with compelling CTAs
- Value proposition cards highlighting key stats
- Product categories preview
- Smooth scrolling navigation
- Mobile-friendly hamburger menu

## Running the Development Server

To view the website locally:

1. Make sure you have Python installed
2. Run the server:
   ```bash
   python server.py
   ```
3. Open your browser and navigate to:
   - http://localhost:8000/
   - http://127.0.0.1:8000/

The server will run on port 8000 by default. Press `Ctrl+C` to stop the server.
